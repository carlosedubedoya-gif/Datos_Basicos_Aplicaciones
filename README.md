# Taller de Estructuras de Datos Básicas

Proyecto de práctica para la asignatura de Estructura de Datos y Algoritmos 2.
Contiene ejemplos y ejercicios interactivos en HTML, CSS y JavaScript para:

- Listas enlazadas
- Pilas
- Colas
- Diccionarios
- Integrador

## Estructura del proyecto

- `templates/`
  - Contiene las páginas HTML del taller
- `static/css/`
  - Estilos visuales usados por las páginas
- `static/js/`
  - Scripts JavaScript que implementan la lógica de cada ejercicio
- `static/imagen/`
  - Imágenes usadas por la interfaz

## Abrir el proyecto

1. Abre el archivo `templates/index.html` en tu navegador.
2. Usa el menú para ir a cada ejercicio.

> Nota: El proyecto no requiere servidor ni compilación. Solo abre el HTML desde el sistema de archivos.
